{% comment %} 경로/섹션 키 단일 로직. - 인자 없이 include: 현재 페이지 기준 url_parts, page_group, section_key, page_group_base 설정. - url 인자로 include: 해당 URL 기준 segs, group_key 설정 (루프에서 section_page.url 넘길 때 사용). 규칙: URL의 "섹션 키" = 첫 번째 경로 세그먼트 (예: /cases/ → cases).
